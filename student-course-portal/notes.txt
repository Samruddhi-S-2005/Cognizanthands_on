Angular Project Notes

1. angular.json
Configuration file for the Angular project. It defines build, serve, test and project settings.

2. tsconfig.json
Main TypeScript configuration shared across the project.

3. tsconfig.app.json
TypeScript configuration specifically for the Angular application.

4. package.json
Contains project information, dependencies and npm scripts.

5. src/main.ts
Entry point of the Angular application. It bootstraps the root application.

6. src/app/app.ts
Root standalone component of the application.

7. src/app/app.config.ts
Application configuration file that registers providers such as routing and services.

8. src/index.html
The main HTML page where the Angular application is loaded.
